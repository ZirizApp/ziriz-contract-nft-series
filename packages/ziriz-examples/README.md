# ZIRIZ Client Implementation Examples

## Overview

An Example implementation of ziriz client using keypair as wallet.

## How to Install

```
npm install // or yarn install
```

## How to run

create .env file contains

```
ADMIN_SECRET=
ADMIN=
```

Admin : Admin stellar address
Admin Secret : Sectret key for admin

command for this project

```
npm run contract:init // initialize contract
npm run contract:create-mint // to create and mint nft
```
